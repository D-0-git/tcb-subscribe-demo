# 云开发实战：开课提醒订阅消息

开发一个在小程序中订阅开课提醒的例子，帮助了解如何基于云开发快速接入小程序订阅消息

## 开发体验配置

- 修改 `project.config.json`中的 appid 为自己小程序的 appid
- 修改 `miniprogram/pages/index`中的 lessTmplID 为自己的订阅消息模板 id
