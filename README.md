## 云开发实战：开课提醒订阅消息

开发一个在小程序中订阅开课提醒的例子，帮助了解如何基于云开发快速接入小程序订阅消息

## 项目结构

- [intact](./intact) 为最终实现的完整版的项目代码
- [init](./init)为原始的项目代码，其中需要开发者按照课程内容自行实现逻辑

## 如何开发和使用

- 使用完整版项目请参考 [intact](./intact)
- 按照初始项目开发请参考 [init](./init)
